# Screenshots Folder

This folder contains screenshots for the README documentation.

## Required Screenshots

Please add the following screenshots to improve the README:

1. **start-screen.png** - The initial game setup screen
2. **gameplay.png** - Active gameplay with multiple guesses
3. **win.png** - Winning screen (4 Bulls)
4. **color-picker.png** - Color selection dialog

## How to Take Screenshots

1. Run the game (`Ex05.exe`)
2. Use **Snipping Tool** or **Win + Shift + S**
3. Save screenshots in this folder with the exact names above
4. Recommended size: 800x600 or higher

## Tips for Good Screenshots

- ✅ Clear, well-lit interface
- ✅ Show interesting game states
- ✅ Crop out unnecessary parts
- ✅ Use PNG format for quality
- ✅ Show different stages of gameplay
